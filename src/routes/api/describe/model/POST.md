# POST /api/describe/:model

Create a new model definition.

## Docs Path

- Live route: `POST /api/describe/:model`
- Docs page: `/docs/api/describe/model/POST`

## Use this page for

- creating a model shell
- adding model metadata
- then adding fields with `POST /api/describe/:model/fields`
